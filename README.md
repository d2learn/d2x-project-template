该项目提供了一个, 可以被[d2x](https://github.com/d2learn/d2x)工具识别和加载的**交互式教程项目**的基础结构. 开发者和教程创作者, 可以基于这个模板来设计开发自己的交互式教程项目

## 功能特色

- 提供了基础的 `Book + Video + Code + X` 结构
- d2x工具可以直接识别和加载
- 提供了项目依赖描述, 支持一键配置环境
